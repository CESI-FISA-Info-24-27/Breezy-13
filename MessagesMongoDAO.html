Class: MessagesMongoDAO

MessagesMongoDAO()

DAO (Data Access Object) pour la gestion des messages avec MongoDB. Fournit des méthodes pour interagir avec la collection "messages" de la base de données.

Constructor

new MessagesMongoDAO()

Initialise la connexion MongoDB et configure la base de données. Les paramètres de connexion sont récupérés depuis les variables d'environnement.
